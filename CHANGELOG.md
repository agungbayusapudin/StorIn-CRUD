## How to Report Issues

Jika Anda menemukan bug atau ingin request feature:

1. Check existing issues terlebih dahulu
2. Buat issue baru dengan template yang disediakan
3. Sertakan:
   - Deskripsi masalah
   - Steps to reproduce
   - Expected behavior
   - Actual behavior
   - Environment info (OS, Go version, etc.)
   - Screenshots/logs jika relevan
